CSCI 3171 - Network Computing

Exercises - Network layer

1. Cite two differences between a router and a switch.

  1. router is a layer-3 device which implemented network layers and below. But switch is a layer-2 device which only implements link layer and below.
  2. router forward packets by IP address but switch forward packets by MAC address.

2. The network layer functionalities can be divided into data plane and control plane. Cite two functions of the data plane and two functions of the control plane.

  1. Data Plane:
    1. Input port functions:using hearder field values, lookup output port using forwarding table in input port memory
    2. Switching fabrics: transfer packet from input link to appropriate output link.

      Or The data plane enables data transfer to and from clients, handling multiple conversations through multiple protocols, and manages conversations with remote peers. Data plane traffic travels through routers, rather than to or from them.

  2. Control Plane:
    1. determines how datagram is routed among routers along end-end path from source host to destination host
    2. Create routing table and forwarding table

3. What are the main components of a network router? Which of these components are implemented in the data plane and which are implemented in the control plane?

  1. Input ports, switching fabric, output ports and routing processor.

  2. Suppose that an arriving packet matches two or more entries in a router’s forwarding
    table. How does the router decide which forwarding rule should be applied?

  3. Compare bus based and interconnection network based switches. Cite at least one
    advantage and one disadvantage for each of them.

  4. Cite two policies for dropping packets when a buffer is full. Which one would you choose
    in a scenario where the network traffic is predominantly from video and web browsing
    applications? Why?

  5. Give an example showing why a network operator might want one class of packets to be
    given priority over another class of packets.

  6. What is the difference between Round Robin (RR) and Weighted Fair Queueing (WFQ)
    packet scheduling policies? Is there a case where RR and WFQ will behave exactly the
    same?

  7. Suppose Host A sends Host B a TCP segment encapsulated in an IP datagram. When
    Host B receives the datagram, how does the network layer in Host B know it should pass the
    segment to TCP rather than to UDP or some other upper-layer protocol?

  8. When a large datagram is fragmented into multiple smaller datagrams at the network
    layer, where are these smaller datagrams reassembled into a single larger datagram?
    (a) the next router
    (b) the last router in the path to the destination
    (c) the receiver
    (d) the packet is reassembled only at the transport layer on the receiver side

  9. How many IP addresses does a router have?

  10. What is meant by the term “route aggregation”? Why is it useful for a router to perform
    route aggregation?

  11. Cite four differences between IPv4 and IPv6. For two of these differences, explain what
    was the main motivation behind them

  12. What is “network neutrality”? Why is it important for Internet users?

  13. Consider a router that interconnects three subnets: Subnet 1, Subnet 2, and Subnet 3.
    Suppose all of the interfaces in each of these three subnets are required to have the prefix
    223.1.17/24. Also suppose Subnet 1 is required to support at least 50 interfaces, Subnet 2 is
    to support at least 80 interfaces, and Subnet 3 is to support at least 10 interfaces. Provide
    three network addresses that satisfy these constraints. OBS.: use CIDR notation.

  14. Consider a subnet with prefix 128.119.40.128/26. Give an example of one IP address
    that can be assigned to this network.

  15. Suppose an ISP owns the block of addresses of the form 128.119.44.64/26. Suppose it
    wants to create four subnets from this block, with each block having the same number of IP
    addresses. What are the prefixes for the four subnets? OBS.: Give your answer using the
    A.B.C.D/X notation.

  16. Cite one advantage and one disadvantage of a “per-router” vs logically centralized
    control plane.

  17. Compare and contrast link-state and distance-vector routing algorithms.

  18. What is the count-to-infinity problem in distance vector routing? How does it affect
    routing on the Internet?

  19. Is it necessary that every autonomous system use the same intra-AS routing algorithm?
    Why or why not?

  20. What is meant by an “area” in OSPF routing? Why was the concept of “area” introduced?

  21. Link-state routing algorithms are based on two phases: link cost flooding and path
    calculation. One can configure these algorithms to flood link cost information periodically or
    based on topology changes. Cite an advantage and a disadvantage from each of these
    approaches.

  22. Cite three criteria that a routing algorithm could use to determine the best paths in a
    network.

  23. What does it mean for the distance vector routing algorithm to be:
    a) iterative
    b) distributed
    c) self-stopping

  24. What are the three main criteria BGP uses to select a route to a given destination prefix?
    In which order are these criteria used?
